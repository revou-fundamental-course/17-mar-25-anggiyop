Luas Persegi

Rumus luas persegi:
L = S x S
Di mana:
L = Luas
S = Sisi

Keliling Persegi

Rumus keliling persegi:
K = 4 x S
Di mana:
K = Keliling
S = Sisi

Luas Persegi Panjang

Rumus luas persegi panjang:
L = p x l
Di mana:
L = Luas
p = Panjang
l = Lebar

Luas Persegi Panjang

Rumus keliling persegi panjang:
K = 2 x ( p + l )
Di mana:
p = Panjang
l = Lebar
